## 🌟 Overview
This is my personal developer portfolio built to showcase my projects, skills, and experience as a **Frontend Developer & MERN Stack Enthusiast**. It highlights my work in modern web development, features an advanced scrollytelling hero background, and provides an elegant, immersive way to connect with me.

## 🚀 Tech Stack
- **Framework:** [Next.js 14](https://nextjs.org/) (App Directory)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** Vercel (https://shiiivxnshh-portfolio.vercel.app/)

## ✨ Features
- 🎬 **Scrollytelling Hero Area:** Image sequence canvas tied directly to your scroll position for cinematic depth.
- 🎨 **Modern UI/UX Aesthetic:** Dark mode defaults, glassmorphism components, neon glow effects, and smooth framer-motion entry animations.
- 📱 **Responsive Design:** Completely optimized layouts tailored for desktops, tablets, and mobile viewing.
- 📁 **Dynamic Showcases:** Interactive Projects, Skills, and Achievements sections equipped with rich detailed modals.
- 📜 **Integrated Document Viewer:** Live Google Drive PDF previews injected directly into framer-motion modals for Certifications and Training tracking.
- ⚡ **High Performance & Typesafety:** Strictly statically-typed server-rendered components and highly optimized payload deliveries.

## 📂 Project Structure
```text
shiiivxnshh-portfolio/
 ├── public/             # Static assets, sequence frames, and images
 ├── src/
 │   ├── app/            # Next.js 14 App Root (page.tsx, layout.tsx, globals.css)
 │   └── components/     # Reusable UI components (Overlay, Navbar, Modals, Sections)
 ├── tailwind.config.ts  # Tailwind CSS configuration and theme tokens
 └── package.json        # Project dependencies and deployment scripts
```

## 🛠️ Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Shivanshdubey09/shiiivxnshh-portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd shiiivxnshh-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the project locally!
